Sublim Text3使用

  • Sublime

1.按照官方说明，安装package control组件

在Console中（Ctrl+~）执行将以下代码

import urllib.request,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

以上安装了package Controller2组件，接下来一个个安装常用插件（Ctrl+Shiht+p）打开控制台，选择package-install，输入相应的插件名称即可
需要安装的插件：

  • Emmet
  • anugularjs
  • nodejs
  • SideBarEnhancements

2.SideBarEnhancements

此插件是增强版的sidebar，可方便使用不同浏览器调试。安装完成后在keybinding-user里输入

[{ "keys": ["f12"], "command": "side_bar_files_open_with", "args": { "paths": [], "application": "C:\\Program Files (x86)\\Google\\Chrome\\Application\\chrome.exe", "extensions":".*" //any file with extension }}]

即可用F12键在Chrome中调试Html

3.Emmet

这个就是传说中的Zencoding，具有代码一键补齐功能

4.html文件一键补齐（现将文件保存为html后缀文件）

输入!或html:5 Tab键,自动生成html文件结构

输入div.row>ui#mainlist>li*5> Tab自动出现
<div class="row"> <ui id="mainlist"> <li></li> <li></li> <li></li> <li></li> <li></li> </ui> </div>

更多Emmet介绍见这里

5.多元素同步编辑

选中要编辑的元素后，CtrlD，则所有相同元素被同步编辑
